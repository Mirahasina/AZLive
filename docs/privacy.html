Politique de confidentialité — AZLive

Dernière mise à jour : juin 2026

1. Données collectées

  • Profil vendeur : nom, contact, identifiants liés TikTok/Facebook (open_id, nom affiché).
  • Données clients : nom, téléphone, adresse, identifiants réseaux sociaux pour les commandes.
  • Données techniques : logs d'utilisation de l'API.

2. Utilisation

Les données servent à authentifier les vendeurs, gérer les lives, capturer et traiter les commandes JP. Nous ne vendons pas vos données à des tiers.

3. TikTok & Facebook

Lorsque vous connectez TikTok ou Facebook, nous accédons uniquement aux informations autorisées par vous (ex. profil public de base). Les tokens d'accès sont stockés de manière sécurisée côté serveur.

4. Conservation

Les données sont conservées tant que votre compte est actif ou selon les obligations légales applicables.

5. Vos droits

Vous pouvez demander l'accès, la correction ou la suppression de vos données en contactant contact@azlive.mg.
